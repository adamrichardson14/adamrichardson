### Install Dependecies

```
npm i eslint eslint-config-next eslint-plugin-import eslint-plugin-react eslint-plugin-sort-keys-fix
```

### Add Linting scripts
```
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint",
    "lint.fix": "next lint --fix"
  },
```

### Ensure code action is to fix all errors on save

```json:/.vscode/settings.json
{
  "workbench.colorCustomizations": {
    "titleBar.activeForeground": "#fff",
    "titleBar.inactiveForeground": "#ffffffcc",
    "titleBar.activeBackground": "#2cff37",
    "titleBar.inactiveBackground": "#2cff37"
  },
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
}

```
